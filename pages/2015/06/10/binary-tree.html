Binary Tree DFS Traversal

三种基本遍历方法

    A
   / \
  B   C
 / \  
D   E

前序(Preorder)：A BDE C

中序(Inorder)：DBE A C

后序(Postorder)：DEB C A

递归 or 非递归？

递归的办法会很简单，一般来说不推荐，然而，Done is better than Perfect！

适用于大多数的Binary Tree Problem

分治法对于二叉树问题，进行如下的操作：

1.Divide：对于左右子树分别去同时处理，将原问题划分成为更小的子问题

2.Conquer：将子问题的解合并，返回问题的解
